# Seguridad

Si encuentras una vulnerabilidad, **no la publiques** en issues.

## Cómo reportar
- Abre un reporte privado mediante los mecanismos de seguridad de GitHub (Security Advisories) si está habilitado
- O contacta al equipo mantenedor (define un correo/alias interno cuando el repo tenga owner)

## Qué incluir
- Pasos para reproducir
- Impacto estimado
- Versiones/commit afectados
- PoC (si aplica) con la mínima información sensible posible

Gracias por ayudar a mantener el proyecto seguro.

# Firebase

Este proyecto incluye dependencias Firebase mediante BoM, pero **no incluye** `google-services.json`.

## 1) Crear proyecto y registrar la app
1. Firebase Console → Create project
2. Add app → Android
3. Package/namespace:
   - `com.masenjo_android.asociacionmayoresvillanueva`
4. Descarga `google-services.json`

## 2) Añadir `google-services.json`
Coloca el fichero aquí:

```text
app/google-services.json
